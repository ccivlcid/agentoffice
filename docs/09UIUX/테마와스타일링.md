## 2. 테마 시스템

### 적용 방식
- `<html data-theme="dark|light">` 속성으로 전환
- localStorage `THEME_STORAGE_KEY`에 저장
- `useTheme()` 훅의 `toggleTheme()` 함수로 토글

### 다크 테마 (기본)
```
배경: 딥 블루 오피스 계열
--th-bg-primary: #0f172a (slate-900)
--th-bg-surface: rgba(30, 41, 59, 0.85)
--th-text-primary: #e2e8f0
--th-border: rgba(71, 85, 105, 0.5)
```

### 라이트 테마

## 7. 반응형 디자인

### 브레이크포인트
| 접두사 | 최소 너비 | 용도 |
|--------|-----------|------|
| (없음) | 0px | 모바일 기본 |
| `sm:` | 640px | 태블릿 |
| `lg:` | 1024px | 데스크탑 |

### 모바일 대응
- 사이드바: `hidden lg:flex` (데스크탑만 표시)
- 모바일 네비게이션: 오버레이 드로어 (`fixed inset-0 z-50`)
- 헤더 버튼: 아이콘 전용 (`sm:hidden` 텍스트 숨김)
- 패딩: `p-3 sm:p-4 lg:p-6`
- 뷰포트: `h-[100dvh]` (다이내믹 뷰포트 높이)

### 레이아웃 구조
```
데스크탑: [Sidebar 240px] [Main: header + content]
모바일:   [Header (햄버거)] → [오버레이 Sidebar] + [Content]
```

## 8. 다국어 (i18n)

### 지원 언어
| 코드 | 언어 | 상태 |
|------|------|------|
| `ko` | 한국어 | 활성 |
| `en` | 영어 | 활성 |
| `ja` | 일본어 | 호환용 (deprecated) |
| `zh` | 중국어 | 호환용 (deprecated) |

### 구현
- `I18nProvider` + `I18nContext`
- `useI18n()` → `{ language, locale, t }`
- 사용법: `t({ ko: "한글", en: "English" })`
- 브라우저 자동 감지 → localStorage 저장
- DB 설정과 동기화

### 이중 언어 헬퍼
```typescript
const tr = (ko: string, en: string) => t({ ko, en })
```
