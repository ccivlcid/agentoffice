## 4. AI 프로바이더 연동

| 프로바이더 | 유형 | 연동 방식 |
|------------|------|-----------|
| **Claude** (Anthropic) | CLI | 네이티브 프로세스 스폰 |
| **Codex** (OpenAI) | CLI | 네이티브 프로세스 스폰 |
| **Gemini** (Google) | CLI | 네이티브 프로세스 스폰 |
| **OpenCode** (GitHub) | CLI | 네이티브 프로세스 스폰 |
| **Copilot** (GitHub) | OAuth | 토큰 교환 + OAuth |
| **Antigravity** (Google) | OAuth | OAuth 토큰 리프레시 |
| **Custom API** | HTTP | OpenAI-호환 API 엔드포인트 |

## 6. 보안 아키텍처

| 기능 | 설명 |
|------|------|
| **세션 쿠키** | `claw_session` 암호화 인증 토큰 |
| **OAuth 토큰 암호화** | AES-256-GCM (`OAUTH_ENCRYPTION_SECRET`) |
| **Origin 검증** | `ALLOWED_ORIGINS`, `ALLOWED_ORIGIN_SUFFIXES` |
| **API 인증 토큰** | `API_AUTH_TOKEN` (원격 접근용) |
| **메시지 감사** | 모든 수신 이벤트 감사 추적 |
| **멱등성 키** | 메시지 중복 처리 방지 |
| **FK 제약조건** | SQLite 참조 무결성 |
