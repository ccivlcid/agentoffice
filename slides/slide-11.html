Getting Started

11

Clone and Install

git clone https://github.com/YOUR_ORG/hyperclaw.git

cd hyperclaw && pnpm install

2

Configure AI Providers

Set up API keys for Claude Code, Codex CLI, Gemini CLI, or other supported providers via the Settings panel.

3

Launch the Office

pnpm dev - Opens your pixel-art CEO office at localhost:5173. Start issuing tasks from the Kanban board!

PREREQUISITES

Node.js 22+ / pnpm / git / At least one AI CLI installed
